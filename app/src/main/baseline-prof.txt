# ART baseline profile rules. The methods listed here are AOT-compiled by
# ProfileInstaller at install time, so they never run interpreted -> JIT on
# cold launch. Each line is an ART-format method or class entry; `H` =
# include in startup pre-compile, `S` = include in startup hot list, `P` =
# include in post-startup. `L` followed by a JVM-style class descriptor
# matches the whole class.
#
# Coverage focus: cold-launch hot path observed in logcat as the JIT
# allocating thousands of KB to compile these methods on first paint. With
# wildcards we don't have to enumerate every method -- ART matches all
# methods whose descriptor matches.
